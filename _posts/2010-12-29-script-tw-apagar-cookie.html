Quando o Tribal Wars ainda não tinha uma equipe para analisar e autorizar os scripts, a idéia de divulgar aqui era para que todos pudessem usar. E agora perdeu um pouco do sentido colocar aqui só para o pessoal usar, já que está lá autorizadão mesmo.

{% include adsense.html %}

Então, já que estou tendo o trabalho de postar aqui, precisa ter algo mais. E o algo mais pode ser o que levou ao script e como cheguei lá.

Este aqui foi porque pediram. E o interessante dele é que encontrei um autorizado no servidor internacional com 38 linhas de código. E é ridículo para algo tão simples. Depois de pensar e pesquisar um pouquinho, criei um com funcionamento idêntico. Porém, como eu poderia dizer isso, é, hum, ele é um pouco menor. Vou mostrar primeiro o código completo:

{% highlight js %} javascript:$.cookie(prompt("Nome do cookie?", "farmeruk"), null);void(0); {% endhighlight %}

Agora o código para usar na barra de acesso rápido:

{% highlight js %} javascript:$.getScript('http://dl.dropbox.com/u/72485850/tribalwarsbrasil/apagar_cookies.js');void(0); {% endhighlight %}

Então..., código ali só é tão pequeno por causa da biblioteca jQuery e plugin jQuery-Cookie. Estas bibliotecas atualmente são usadas pelo Tribal Wars. Algumas pessoas parecem ter resistência em utilizar o que está disponível, preferindo usar JavaScript "puro". Outro incluem no código vários testes para situações hipotéticas que nunca vão ocorrer. Posso citar um exemplo no próprio link que estpa ali no código para usar na barra de acesso rápido, incluiram um teste se o jogador tem conta premium que é totalmente desnecessário.

Eu prezo sempre pelo menor código, desde que legível, fácil de entender, limpo e sem complicações desnecessárias.
