Muitas tarefas para fazer, pouco tempo para realizar. Situação comum, acha? Então some a preguiça e procrastinação. Já tentei muita coisa, Task Coach e w2do foram os melhores. Sempre combinados com o Dropbox. Mas como sempre acontece, alguma coisa com os programas prontos sempre me chateia. O Task Coach é em Python e consequentemente desnecessariamente grande. Mas o problema mesmo é que não funciona no computador do trabalho, onde preciso. O w2do também tem um problema semelhante, é em Perl. Outra coisa é que, apesar que gosto de usar a linha de comando, fazer isso no Windows é chato. Então, apresento-lhes:

Simples de usar, você digita para pesquisar nas tarefas existente. Use ENTER para começar a criar uma nova tarefa. Você pode colocar uma data, comentário, tag, além de configurar tarefas recorrentes. Para criar novas tags use F10. Os botões com bolinhas coloridas servem para você marcar se deseja ou não exibir as tarefas atrasadas (vermelho - mais de 1 dia em atraso), tarefas de ontem (violeta), para hoje (laranja), para amanhã (azul), ou futuras (cinza - mais de 1 dia a frente). Você também pode trabalhar com múltiplos banco de dados (ou banco de tarefas). Por falar em banco de dados, é com SQLite. Bom, é isso. Simples e direto. O foco é em você realizar as tarefas, não em centenas de funcionalidades.
