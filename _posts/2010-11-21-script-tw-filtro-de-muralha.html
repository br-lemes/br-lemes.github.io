Reconstruir muralha é uma tarefa constante e enfadonha. Se descuidarmos perdemos tropas de apoio à toa por falta de refazer as muralhas. Tínhamos o Filtro de Edifícios do Relaxeaza, mas este aqui é mais rápido - e tempo é essencial antes que cheguem os novos ataques.

Funciona em Visualizações - Edifícios. Desparece as aldeias com muralha completa (nível 20) e ficam as aldeias com muralha incompleta. Não precisa configurar nada, só clicar e subir as muralhas.

Script para colar na barra de acesso rápido:

{% highlight js %} javascript:$.getScript('http://dl.dropbox.com/u/72485850/tribalwarsbrasil/filtro_de_muralha.js');void(0); {% endhighlight %}

Script completo (que nem é muito maior):

{% highlight js %} $('td [class="upgrade_building b_wall"]').each(function(i,e){ if ($(e).text()==20) e.parentNode.parentNode.removeChild(e.parentNode); }); {% endhighlight %}
