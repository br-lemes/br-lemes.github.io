Se você gosta de deixar todas as suas aldeias com tudo pesquisado, quando tiver muitas aldeias vai dar um trabalhinho encontrar aldeias que ainda possuem pesquisas faltando. Este script vai lhe facilitar a vida, use em Visualizações - Pesquisa.

{% highlight js %} javascript:$.getScript('http://dl.dropbox.com/u/72485850/tribalwarsbrasil/filtro_de_pesquisa_simples.js');void(0); {% endhighlight %}
{% include adsense.html %}

Ao clicar nele você vai ver apenas as aldeias que tem alguma pesquisa faltando, qualquer uma que seja.

A diferença entre este e o Filtro de pesquisas do Relaxeaza, é que este você não precisa escolher qual pesquisa deseja filtrar, ele simplesmente filtra aldeias que faltam qualquer uma das pesquisas.

Escrevi este script para mundos com ferreiro simples, não importa se é com ou sem arqueiro, não sei como são mundos com ferreiro por níveis.
